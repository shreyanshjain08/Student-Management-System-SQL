# Student Management System using SQL

## Description
This project demonstrates a relational database solution for managing student academic data. It focuses on database design, relationships, and optimized SQL queries.

## Features
- Normalized database schema
- Use of primary and foreign keys
- Joins and aggregate functions
- Real-world business queries

## Technologies Used
- SQL (MySQL)
- GitHub

## Use Cases
- Student performance analysis
- Course-wise reporting
- Academic record management
